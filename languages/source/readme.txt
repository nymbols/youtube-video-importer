This is the default translation file. If you would like to help translating this plugin in your language, there are a few things you will need first:

1. Download and install Poedit (http://www.poedit.net/download.php)

2. After installation, open file default.po with Poedit.

3. When saving your translation, Poedit will automatically create a .mo file.

4. After finishing all translations, e-mail both file to support [at] youtubeimporter.com Please use as subject "YouTube WordPress plugin translation" and don't forget to specify the language you translated in.

Thank you for contributing to the translation process of this plugin.
